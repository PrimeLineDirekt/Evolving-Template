## Anti-Patterns (What NOT to Do)

### ❌ Keyword Stuffing Title
```
Coffee Kitchen Print Wall Art Poster Digital Download Farmhouse Rustic Typography
```

### ❌ Too Long German Tags
```
sonnenuntergang-am-strand-aquarell (32 chars - TOO LONG!)
→ Fixed: strand-sunset (13 chars)
```

### ❌ Wrong Version Selection
```
Baby Room with --v7  ← Should use --v6 for softer look
Office art with --v6 ← Should use --v7 for modern feel
```

### ❌ Pinterest Title Too Long
```
Serene Sunset Beach Watercolor Print Perfect for Coastal Living Room Home Decor (80 chars)
→ Fixed: Sunset Beach Watercolor - Coastal Living Room (48 chars)
```
