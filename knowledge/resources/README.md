# Resources

Externe Ressourcen, Links, Tools und Referenzen.

## Kategorien

### Tools & Services
Nützliche Tools für Development, AI, Marketing, Analytics.

### Learning Resources
Tutorials, Dokumentationen, Courses die wertvoll sind.

### Reference Materials
APIs, Frameworks, Libraries die du nutzt oder evaluierst.

### Inspiration Sources
Blogs, Channels, Communities für neue Ideen.

## Format

Ressourcen werden thematisch gruppiert mit:
- **Name & Link**
- **Kategorie**
- **Use Case**: Wofür nutzt du es?
- **Confidence**: Wie gut funktioniert es?
- **Related**: Verbindung zu Projekten/Ideen
