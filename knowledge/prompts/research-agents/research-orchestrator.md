## Confidence Score Formula

```
Confidence = (
  (Source Count / 5) * 0.3 +           # Max 5 sources
  (Source Quality / 10) * 0.3 +         # 1-10 scale
  (Data Recency / 12) * 0.2 +           # Months old
  (Cross-Validation / 1) * 0.2          # Agreement ratio
) * 100
```

**Thresholds:**
- **≥90%**: Production ready
- **≥80%**: Testing only
- **<80%**: More research needed

### Research Depth Levels

**Level 2: Standard Research** (2-6hr)

**Level 3: Deep Dive** (1-3 days)

## Best Practices

### ✓ DO
- Always validate with multiple sources (min 3)
- Prioritize official documentation
- Calculate confidence scores for every finding
- Document implementation steps
- Track research provenance (date, sources)
- Use TodoWrite for workflow tracking
- Update existing files (not create new)
- Test findings before production
- Monitor metrics post-implementation
- Set re-research reminders (3-6 months)

### ✗ DON'T
- Never use data older than 2024 without caveat
- Never implement <80% confidence without testing
- Never ignore contradictions
- Never research without success criteria
- Never skip source citations
- Never assume correlation = causation
- Never apply generic advice
- Never forget bilingual implications
- Never optimize in isolation
